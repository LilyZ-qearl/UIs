## 属性 Attributes

| 参数  | 说明  | 类型   | 可选值           | 默认值      |
|:------|:-----------------|:-------|:---------------------|:------------|
| outWidth | 伸缩区域的外框宽度 | Number | ---- | 1366 |
| outHeight | 伸缩区域的外框高度 | Number | ---- | 768 |
| width | 随着外框区域变化而自适应变化的宽度 即展示区域的宽度 | Number | ---- | 1366 |
| height | 随着外框区域变化而自适应变化的高度 即展示区域的高度 | Number | ---- | 768 |
