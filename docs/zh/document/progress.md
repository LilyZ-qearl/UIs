## 属性 Attributes

| 参数  | 说明  | 类型   | 可选值           | 默认值 |
|:------|:-------------|:-------|:------------------|:--------|
| percent | 百分比 | Number / String | 0-100 | 0 |
| strokeWidth | 进度条宽度 | Number / String | ---- | 6 |
| length | 进度条长度 | Number / String | ---- | 200 |
| status | 状态 | String | 成功 / 异常 | ---- |
